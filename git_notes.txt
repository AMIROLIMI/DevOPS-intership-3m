Мои заметки по Git

ветки (branch)
ветки позволяют работать над разными частями проекта одновременно. Это полезно для разработки новых функций, 
исправления ошибок, не затрагивая основную ветку.

Основные команды:
- git branch amir: Создание новой ветки.
- git checkout amir: Переключение на другую ветку.
- git merge amir: Слияние изменений из одной ветки в другую.
- git branch -d amir: Удаление ветки.
